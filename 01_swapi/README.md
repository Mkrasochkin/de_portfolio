# Проект SWAPI

## Описание проекта

SWAPI или Star Wars API хранит информацию о персонажах, космических кораблях, планетах и других объектах, которые связаны со вселенной «Звёздных войн». Предстоит поработать с этим API: отправить к нему несколько запросов и разобрать его ответы.

## Навыки и инструменты

python, requests, ООП


Обработка исключений: применение методов перехвата и обработки ошибок, возникающих при выполнении запросов.


Проверка статуса ответа сервера: проверка успешности запросов с использованием метода raise_for_status.

Работа с файлами: сохранение полученных данных в файлы формата .txt.

Модульное проектирование: разделение функционала на классы и методы для повышения читаемости и удобства поддержки кода.

## Общий вывод

В рамках проекта была реализована работа с внешним API. Данный проект демонстрирует базовые техники работы с внешним API и применённые принципы объектно-ориентированного программирования (ООП). Разработаны специализированные классы для взаимодействия с API, обеспечивающие удобное управление запросами и получение необходимой информации.
Реализация основана на библиотеке requests для отправки HTTP-запросов, обработке возможных ошибок и проверке состояний ответов сервера. 
Опыт, приобретённый в процессе выполнения проекта, полезен для дальнейшей работы с интеграцией сторонних сервисов и API, позволяя строить надёжные и эффективные системы сбора и анализа данных.

```shell script
python3 -m venv venv
```

* Активируйте его:
```shell script
source venv/bin/activate
```
или в Windowns
```shell script
source venv/Scripts/activate
```

* Обновите pip до последней версии:
```shell script
pip install --upgrade pip
```
* Установите зависимости:
```shell script
pip install -r requirements.txt
```
* После выполнения задания убедитесь что успешно пройдены все тесты
