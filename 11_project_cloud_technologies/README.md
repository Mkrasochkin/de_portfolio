# Проект по облачному хранилищу данных


## Описание проекта

Проект описывает процесс построения облачного хранилища данных (DWH) с использованием микросервисной архитектуры и инструментов Yandex Cloud. Система собирает данные из Kafka, нормализует их и выводит в виде аналитических витрин.

## Архитектура и рабочий процесс


![Архитектура проекта](https://github.com/Mkrasochkin/de_portfolio/blob/main/11_project_cloud_technologies/2_1747338249.png) 


1. **STG Service**: Прием сырой информации из Kafka и помещение в PostgreSQL.
2. **DDS Service**: Распределение нормализованных данных по таблицам в слое DDS.
3. **CDM Service**: Генерация аналитических витрин на основе данных из DDS.
4. **DataLens**: Визуализация результатов на финальном этапе.


## Навыки и инструменты

- Язык программирования: Python.
- Базовая работа с контейнерами: Docker, Kubernetes.
- Основы стриминга: Apache Kafka.
- Облачные инструменты: Yandex Cloud.

## Общий вывод

Проект позволил освоить базовые концепции и инструменты дата-инженерии. Был пройден путь от приема данных из Kafka до их отображения в виде простых аналитических отчетов. Освоены принципы работы с многослойными архитектурами и автоматизированной доставкой данных.
