# Автосалон «Бибип»

## Описание проекта

«Бибип» — новый быстроразвивающийся салон по продаже автомобилей. IT-отдел салона принял решение не пользоваться популярными СУБД, а создать свой аналог: данные хранить в файлах, а обрабатывать их с помощью Python. Задача — создать базу данных, которую будут использовать приложения «Бибип». Организовать хранение данных в файлах с индексами для быстрой обработки и минимальной нагрузки на память. Реализовать функциональность для добавления, удаления и изменения записей о машинах, моделях и продажах. Осуществить аналитику данных для выдачи актуальной информации о доступности автомобилей, изменениях статусов заказов и общей статистике продаж.

## Навыки и инструменты

* **Python:** реализация файлового аналога СУБД, CRUD-операции, аналитические запросы.  
* **CRUD:** создание, чтение, обновление и удаление данных.  
* **Индексация:** создание и поддержание индексов для быстрого доступа к данным.  
* **Аналитика:** выборка актуальных данных о доступности автомобилей, изменении статусов заказов и статистике продаж.  
* **Эффективность:** минимизация потребления памяти и оптимизация операций.

## Общий вывод

В ходе проекта была разработана легкая и производительная файловая база данных для автосервиса на Python. Проект позволяет не просто хранить записи, но и извлекать из них аналитику: от детальной карточки автомобиля до формирования ТОП-3 самых прибыльных моделей для отдела маркетинга. Это позволило сотрудникам и клиентам эффективно работать с информацией о машинах, заказах и продажах.
